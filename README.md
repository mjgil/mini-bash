# mini-bash

bash functions for all users of computer/server to use

**Linux:**
```bash
bash <(wget -qO- https://raw.githubusercontent.com/mjgil/mini-bash/master/install-global.sh)
```

**macOS:**
```bash
bash <(curl -fsSL https://raw.githubusercontent.com/mjgil/mini-bash/master/install-global.sh)
```

**macOS note:** For full `sort -h` (human-readable) support in `dfg` and `dug`, install GNU coreutils:
```bash
brew install coreutils
```
